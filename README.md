## 📌 Description

**ShieldTrip** is a smart travel safety application designed to help users navigate safely using real-time alerts, safe route suggestions, and location-based risk visualization.

Originally conceptualized during **SIH 2025**, the project focuses on enhancing user safety while traveling—especially in unfamiliar or high-risk areas.

## 📏 Design Guidelines

* Keep UI clean, minimal, and intuitive  
* Focus on **user safety & clarity**  
* Use reusable components  
* Maintain consistency across all screens  

---
### 🎨 Typography Rules (STRICT)
* Use **Poppins** font only (no alternatives)  
* Follow consistent font sizes and hierarchy  
* Do not introduce new font styles or random variations  
* Maintain proper spacing and readability
